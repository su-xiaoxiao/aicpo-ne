# 文档模板目录说明

本目录存放当前仓库各类设计文档的标准模板。

使用原则：

- 新建正式文档时优先从模板复制
- 模板保留固定结构，不写具体业务内容
- 同类型文档尽量保持统一章节顺序
- 若某模板不适用，应在具体文档中删除无关章节，不要新增杂乱结构

当前模板包括：

- `00-背景文档模板.md`
- `01-治理文档模板.md`
- `02-领域文档模板.md`
- `03-架构文档模板.md`
- `10-模块设计文档模板.md`
- `20-概念模型模板.md`
- `21-逻辑模型模板.md`
- `22-物理模型模板.md`
- `30-模型图说明模板.md`
- `40-评审记录模板.md`
